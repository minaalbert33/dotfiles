# Mina dotfiles

This is the configuration of my Arch linux installation.
I'm using the Hyprland (Wayland).

## Common Packages

- Terminal: alacritty
- Editor: vim, visual studio cod
- Icons: Font Awesome, JetBrains Mono
- Menus: Rofi (dmenu Theme)
- Browsers: brave
- Filemanager: ranger, dolphin

## Hyprland

- Status Bar: waybar
- Screenshots: hyprshot
- Wallpapers: hyprpaper
